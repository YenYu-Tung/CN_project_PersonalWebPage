# CN_project - personal page
## features
* signup/login/logout
* video streaming
* music streaming
* bonus
1. encryped passwords 
2. todo-list
## environment setup
* install node.js & npm
* clone the repository to local
* $ npm install
* $ npm run start
* see "Server is running!" means run successfully
* ctrl + c to stop server
## development tools
* Node.js 16.5.0
* Express 4.17.1
* Express-Handlebars 4.0.2
* Bootstrap 5.1.3
* Font-awesome 5.6.3
